       ```sql
       CREATE TABLE IF NOT EXISTS advisor_budget (
         agent TEXT NOT NULL,
         date TEXT NOT NULL,
         calls_used INTEGER NOT NULL DEFAULT 0,
         max_calls INTEGER NOT NULL DEFAULT 10,
         PRIMARY KEY (agent, date)
       );
       ```
    2. Export `AdvisorBudget` class with constructor taking `db: Database` (better-sqlite3)
    3. Methods (all use prepared statements):
       - `canCall(agent: string): boolean` -- check today's row, return calls_used < max_calls. If no row for today, return true (fresh day)
       - `recordCall(agent: string): void` -- INSERT OR REPLACE with calls_used + 1 for today's date
       - `getRemaining(agent: string): number` -- return max_calls - calls_used for today (default 10 if no row)
    4. Use `new Date().toISOString().slice(0, 10)` for today's date string (YYYY-MM-DD)

    2. Validate `agent` and `question` string params
    3. Check budget via `advisorBudget.canCall(agent)` -- throw ManagerError if exhausted
