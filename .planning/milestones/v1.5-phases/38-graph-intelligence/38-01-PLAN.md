    **Step 1: Create `src/memory/graph-search.types.ts`**

    Define these frozen/readonly types:

    ```typescript
    export type GraphSearchConfig = {
      readonly neighborSimilarityThreshold: number; // default 0.3
      readonly maxNeighbors: number;                // default 5
      readonly maxTotalResults: number;             // default 15
    };

    export type GraphSearchResult = {
      readonly id: string;
      readonly content: string;
      readonly relevanceScore: number;
      readonly combinedScore: number;
      readonly tags: readonly string[];
      readonly createdAt: string;
      readonly source: "knn" | "graph-neighbor";

    };
    ```

    Also export a `DEFAULT_GRAPH_SEARCH_CONFIG` constant with threshold 0.3, maxNeighbors 5, maxTotalResults 15.

    **Step 2: Create `src/memory/graph-search.ts`**

    Create a `GraphSearch` class that:
    1. Constructor takes `MemoryStore` and optional `Partial<GraphSearchConfig>` (merges with defaults)
    2. Has a `search(queryEmbedding: Float32Array, topK: number): readonly GraphSearchResult[]` method
    3. Search algorithm:
       a. Create `SemanticSearch` from `store.getDatabase()`, call `.search(queryEmbedding, topK)` to get KNN results
       b. Collect KNN result IDs into a `Set<string>`
       c. For each KNN result, call `getForwardLinks(store, result.id)` and `getBacklinks(store, result.id)`
       d. For each neighbor not in knnIds, fetch its embedding via `store.getEmbedding(neighborId)`
       e. Compute cosine similarity between neighbor embedding and `queryEmbedding` using a `cosineSimilarity(a, b)` utility (dot product of L2-normalized vectors)
       f. Only include neighbors with similarity >= `config.neighborSimilarityThreshold`
       g. Track which KNN result(s) link to each neighbor in a `linkedFrom` map
       h. Sort eligible neighbors by similarity descending, take top `config.maxNeighbors`
       i. Map KNN results to `GraphSearchResult` with `source: "knn"`, no `linkedFrom`
       j. Map neighbor results to `GraphSearchResult` with `source: "graph-neighbor"`, `linkedFrom` array
       k. Concatenate KNN results + neighbor results, cap at `config.maxTotalResults`
       l. Freeze everything per project convention

    Also export `cosineSimilarity(a: Float32Array, b: Float32Array): number` as a named export (dot product -- embeddings are L2-normalized).

    IMPORTANT: Do NOT mutate SemanticSearch. Compose around it. Do NOT use `store.getById()` for neighbors (it increments access_count). Use `getForwardLinks`/`getBacklinks` which return MemoryEntry without side effects.
