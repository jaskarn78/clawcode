    **Step 1: Create `src/memory/similarity.ts`**

    Export a `cosineSimilarity(a: Float32Array, b: Float32Array): number` function that computes the dot product of two L2-normalized vectors. Since all embeddings from EmbeddingService are already normalized, dot product = cosine similarity.

    ```typescript
    export function cosineSimilarity(a: Float32Array, b: Float32Array): number {
      let dot = 0;
      for (let i = 0; i < a.length; i++) {
        dot += a[i] * b[i];
      }
      return dot;
    }
    ```

    Export a `discoverAutoLinks` function with this signature:

    ```typescript
    type AutoLinkConfig = {
      readonly similarityThreshold: number; // default 0.6
      readonly batchSize: number;           // default 50
    };

    type AutoLinkResult = {
      readonly linksCreated: number;
      readonly pairsScanned: number;
      readonly skippedExisting: number;
    };

    export function discoverAutoLinks(
      store: MemoryStore,
      config?: Partial<AutoLinkConfig>,
    ): AutoLinkResult;
    ```

    Algorithm for `discoverAutoLinks`:
    1. Merge config with defaults: `{ similarityThreshold: 0.6, batchSize: 50 }`
    2. Query recent memories that have NO outbound auto-created edges. Use a raw SQL query on `store.getDatabase()`:
       ```sql
       SELECT m.id FROM memories m
       WHERE m.tier != 'cold'
         AND NOT EXISTS (
           SELECT 1 FROM memory_links ml
           WHERE ml.source_id = m.id AND ml.link_text = 'auto:similar'
         )
       ORDER BY m.created_at DESC
       LIMIT ?
       ```
    3. For each candidate memory, get its embedding via `store.getEmbedding(id)`. Skip if null.
    4. For each candidate, use KNN search (via sqlite-vec) to find top-5 similar memories. Use a raw query:
       ```sql
       SELECT memory_id, distance FROM vec_memories
       WHERE embedding MATCH ? AND k = 6
       ORDER BY distance
       ```
       (k=6 to account for self-match, then filter out self)
    5. For each KNN neighbor of a candidate:
       - Skip if same ID as candidate
       - Compute cosine similarity from embedding (or convert distance via `1 - distance`)
       - Skip if below `config.similarityThreshold`
       - Check if edge already exists (either direction) using:
         ```sql
         SELECT 1 FROM memory_links
         WHERE (source_id = ? AND target_id = ?) OR (source_id = ? AND target_id = ?)
         ```
       - If no edge exists, create BIDIRECTIONAL edges:
         - `insertLink(candidateId, neighborId, "auto:similar", now)`
         - `insertLink(neighborId, candidateId, "auto:similar", now)`
       - Increment counters
    6. Return `{ linksCreated, pairsScanned, skippedExisting }` as a frozen object

    IMPORTANT: All prepared statements should be created once and reused. Wrap the function body in a transaction for atomicity. Use `store.getDatabase().transaction(...)`.

    **Step 2: Create `src/heartbeat/checks/auto-linker.ts`**

    Follow the consolidation check pattern exactly:

    ```typescript
    import type { CheckModule, CheckResult } from "../types.js";
    import { discoverAutoLinks } from "../../memory/similarity.js";

    const runningAgents = new Set<string>();

    const autoLinkerCheck: CheckModule = {
      name: "auto-linker",
      interval: 21600, // 6 hours
      timeout: 60,     // 1 minute

      async execute(context): Promise<CheckResult> {
        const { agentName, sessionManager } = context;

        if (runningAgents.has(agentName)) {
          return { status: "healthy", message: "Auto-linker already running, skipping", metadata: { skipped: true } };
        }

        const memoryStore = sessionManager.getMemoryStore(agentName);
        if (!memoryStore) {
          return { status: "warning", message: `Memory store not found for ${agentName}` };
        }

        runningAgents.add(agentName);
        try {
          const result = discoverAutoLinks(memoryStore);
          return {
            status: "healthy",
            message: `Auto-linker: created ${result.linksCreated} links, scanned ${result.pairsScanned} pairs, skipped ${result.skippedExisting} existing`,
            metadata: { ...result },
          };
        } finally {
          runningAgents.delete(agentName);
        }
      },
    };

    export default autoLinkerCheck;
    ```

    Note: `discoverAutoLinks` is synchronous (all SQLite operations are sync via better-sqlite3), but the heartbeat check wraps it in an async execute method. This is fine -- the function returns immediately.
