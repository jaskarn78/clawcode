  <behavior>
    - autoLinkMemory(store, memoryId) returns AutoLinkResult with linksCreated, pairsScanned, skippedExisting
    - Creates bidirectional edges (source->target AND target->source) with link_text 'auto:similar'
    - Skips self-match from KNN results
    - Skips cold-tier neighbors
    - Skips already-linked pairs (either direction)
    - Returns {linksCreated: 0, pairsScanned: 0, skippedExisting: 0} when memory has no embedding
    - Uses similarity threshold 0.6 (same default as discoverAutoLinks)
    - Accepts optional Partial<AutoLinkConfig> for threshold override
  </behavior>
  <action>
    Add `autoLinkMemory(store: MemoryStore, memoryId: string, config?: Partial<AutoLinkConfig>): AutoLinkResult` to `src/memory/similarity.ts`.

    Implementation approach — extract the per-memory logic from `discoverAutoLinks` into `autoLinkMemory`:
    1. Get the embedding for `memoryId` via `store.getEmbedding(memoryId)`. Return zero-result if null.
    2. Get the database via `store.getDatabase()`.
    3. Run KNN search: `SELECT memory_id, distance FROM vec_memories WHERE embedding MATCH ? AND k = 6 ORDER BY distance` with the memory's embedding.
    4. For each neighbor: skip self, skip cold-tier (`SELECT tier FROM memories WHERE id = ?`), convert cosine distance to similarity (`1 - distance`), skip below threshold.
    5. Check existing edge in either direction: `SELECT 1 FROM memory_links WHERE (source_id = ? AND target_id = ?) OR (source_id = ? AND target_id = ?)`.
    6. If no edge exists, insert bidirectional: `INSERT OR IGNORE INTO memory_links (source_id, target_id, link_text, created_at) VALUES (?, ?, 'auto:similar', ?)` for both directions.
    7. Wrap all DB operations in a transaction.
    8. Return frozen AutoLinkResult.

    Export `autoLinkMemory` from the module.

    - A cold-tier memory is NOT linked
    - An already-linked pair is skipped (skippedExisting incremented)
    - A memory with no embedding returns zero-result
    - The function does not affect the heartbeat auto-linker (discoverAutoLinks still works independently)
