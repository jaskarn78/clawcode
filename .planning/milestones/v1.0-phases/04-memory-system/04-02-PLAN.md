       - Type `CompactionDeps` (dependency injection):
         - `memoryStore: MemoryStore`
         - `embedder: EmbeddingService`
         - `sessionLogger: SessionLogger`
         - `threshold: number` (0-1, from config)
         - `log: Logger` (pino)
       - Type `ConversationTurn`:
         - `readonly timestamp: string`
         - `readonly role: 'user' | 'assistant'`
         - `readonly content: string`
       - Type `ContextFillProvider`:
         - `getContextFillPercentage(): number` — returns 0-1. Abstracted because SDK may not expose this (Pitfall 5 from research). Default implementation: character count heuristic.
       - Class `CompactionManager`:
         - Constructor takes `CompactionDeps`
         - `shouldCompact(fillPercentage: number): boolean` — returns true if fillPercentage >= threshold. Per D-16.
         - `async compact(conversation: readonly ConversationTurn[], extractMemories: (text: string) => Promise<readonly string[]>): Promise<{logPath: string, memoriesCreated: number, summary: string}>`:
           1. Flush conversation to daily log via sessionLogger.flushConversation() — per D-04/D-17 step 1
           2. Record session log entry in store via store.recordSessionLog()
           3. Call extractMemories callback with full conversation text — per D-18 (agent extracts facts)
           4. For each extracted memory: embed via embedder.embed(), insert via store.insert() with source='conversation'
           5. Generate summary (concatenate last N facts as context summary)
           6. Return { logPath, memoriesCreated, summary }
         - The `extractMemories` callback is intentionally a parameter, NOT hardcoded — the agent itself does the extraction per D-18. The SessionManager will provide this callback by asking the agent to identify key facts.
       - Create a `CharacterCountFillProvider` class implementing `ContextFillProvider`:
         - Tracks total characters from conversation turns
         - `addTurn(content: string): void`
         - `getContextFillPercentage(): number` — returns characters / maxCharacters (configurable, default 200000 as rough proxy for Claude's context)
         - This is the heuristic fallback per research open question 1

    4. Update `src/memory/index.ts`:
       - Export CompactionManager, CompactionDeps, ConversationTurn, ContextFillProvider, CharacterCountFillProvider
