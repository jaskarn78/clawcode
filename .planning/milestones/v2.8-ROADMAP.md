# Roadmap: ClawCode v2.8 — Performance + Reliability

**Theme:** Lock in the Phase 110 RSS savings, then attack tool-call latency, subagent reliability, and operator UX. Bundles 14 backlogged 999.x items + 1 in-flight phase + 4 unscheduled standalones into a coherent milestone.

**Status:** PROPOSED 2026-05-07. Pending operator approval before opening phase plans.

**Constraint:** Zero new npm deps (matches v2.7 discipline).
