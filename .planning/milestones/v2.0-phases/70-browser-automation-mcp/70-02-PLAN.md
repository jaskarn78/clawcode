<objective>
Wave-2 surface for Phase 70: implement the 6 browser tool handlers as PURE functions over `BrowserContext`, add the Readability + screenshot envelope helpers, wire them into a stdio MCP subprocess (`clawcode browser-mcp`) that IPCs back to the daemon-owned `BrowserManager`, and declare the `browser-tool-call` IPC contract that Plan 03 will wire on the daemon side.

Purpose: This plan delivers BROWSER-01..05 tool surface against the `BrowserManager` built in Plan 01. The separation of `tools.ts` (pure) from `mcp-server.ts` (MCP+IPC glue) from `cli/commands/browser-mcp.ts` (CLI entrypoint) is deliberate:
- `tools.ts` is unit-tested against fake BrowserContext — fast, no Playwright required.
- `mcp-server.ts` is unit-tested with an in-memory MCP client + mocked `sendIpcRequest`.
- Real Playwright runs only in an optional `tools.integration.test.ts` guarded by a `PLAYWRIGHT_INSTALLED` env flag (smoke-tested by Plan 03's `scripts/browser-smoke.mjs`).

The MCP subprocess architecture follows 70-RESEARCH.md Open Question #3 resolution: new `clawcode browser-mcp` CLI subcommand (parallel to the existing `clawcode mcp` in `src/cli/commands/mcp.ts`) that starts a `StdioServerTransport` and forwards every tool call to the daemon over Unix socket IPC. The daemon owns the singleton `BrowserManager`; the MCP subprocess is a thin translator.

    1. Create `src/browser/readability.ts`:
       ```typescript
       import type { BrowserLogger } from "./types.js";

       export interface ArticleResult {
         readonly title: string | null;
         readonly byline: string | null;
         readonly siteName: string | null;
         readonly publishedTime: string | null;
         readonly lang: string | null;
         readonly excerpt: string | null;
         readonly text: string;
         readonly html: string;
         readonly length: number;
       }

       /**
        * Parse rendered HTML through @mozilla/readability. Returns null on pages
        * Readability cannot identify as articles. Lazy-imports both jsdom and
        * readability so this module is cheap to import even when unused.
        */
       export async function parseArticle(
         html: string,
         baseUrl: string,
         log?: BrowserLogger,
       ): Promise<ArticleResult | null> {
         const { JSDOM } = await import("jsdom");
         const { Readability } = await import("@mozilla/readability");
         const dom = new JSDOM(html, { url: baseUrl });
         try {
           const reader = new Readability(dom.window.document);
           const article = reader.parse();
           if (!article) return null;
           const text = (article.textContent ?? "").replace(/\s+/g, " ").trim();
           return Object.freeze({
             title: article.title ?? null,
             byline: article.byline ?? null,
             siteName: article.siteName ?? null,
             publishedTime: article.publishedTime ?? null,
             lang: article.lang ?? null,
             excerpt: article.excerpt ?? null,
             text,
             html: article.content ?? "",
             length: text.length,
           });
         } catch (err) {
           log?.warn({ err: (err as Error).message }, "readability parse threw");
           return null;
         } finally {
           dom.window.close();
         }
       }
       ```

       import { join } from "node:path";
       import { format } from "date-fns";
       import { nanoid } from "nanoid";

       export type McpContent =
         | { readonly type: "text"; readonly text: string }
         | { readonly type: "image"; readonly data: string; readonly mimeType: string };

       export interface ScreenshotEnvelope {
         readonly content: readonly McpContent[];
       }

       export function encodeScreenshot(
         buffer: Buffer,
         maxInlineBytes: number,
         meta: { path: string },
       ): ScreenshotEnvelope {
         const bytes = buffer.length;
         const shouldInline = maxInlineBytes > 0 && bytes <= maxInlineBytes;
         if (shouldInline) {
           return Object.freeze({
             content: Object.freeze([
               Object.freeze({ type: "text" as const, text: JSON.stringify({ path: meta.path, bytes, inline: true }) }),
               Object.freeze({ type: "image" as const, data: buffer.toString("base64"), mimeType: "image/png" }),
             ]),
           });
         }
         return Object.freeze({
           content: Object.freeze([
             Object.freeze({
               type: "text" as const,
               text: JSON.stringify({
                 path: meta.path,
                 bytes,
                 inline: false,
                 note: "Screenshot too large to inline; use Read tool on the path.",
               }),
             }),
           ]),
         });
       }

       export function resolveScreenshotSavePath(
         workspace: string,
         overridePath?: string,
       ): string {
         if (overridePath && overridePath.length > 0) return overridePath;
         const date = format(new Date(), "yyyy-MM-dd");
         const id = `${Date.now()}-${nanoid(6)}`;
         return join(workspace, "browser", "screenshots", date, `${id}.png`);
       }
