    New PolicyResult:
    ```typescript
    export type PolicyResult =
      | Readonly<{ allow: true; targetAgent: string; payload: string; ruleId: string }>
      | Readonly<{ allow: false; reason: string }>;
    ```

    PolicyEvaluator class:
    ```typescript
    export class PolicyEvaluator {
      private readonly rules: readonly CompiledRule[];
      private readonly throttles: Map<string, TokenBucket>;
      private configuredAgents: ReadonlySet<string>;

      constructor(rules: readonly CompiledRule[], configuredAgents?: ReadonlySet<string>);
      evaluate(event: TriggerEvent): PolicyResult;
      updateConfiguredAgents(agents: ReadonlySet<string>): void;
    }
    ```

    evaluate() logic:
    1. Filter to enabled rules only
    2. Iterate rules in order (already sorted by priority desc from loader)
    3. For each rule, call `matchesSource(rule.source, event)` — hand-rolled glob matching:
       - `globMatch(pattern, value)`: if pattern === "*" return true; if pattern.endsWith("*") return value.startsWith(pattern.slice(0,-1)); else return pattern === value
       - `matchesSource()`: if rule.kind set and event.sourceKind set, glob-match kind; if rule.id set, glob-match sourceId; both must pass
    4. First matching rule: check throttle (if rule.throttle, call TokenBucket.tryConsume())
    5. If throttled: return `{ allow: false, reason: "rule '${rule.id}' throttled" }`
    6. Check configuredAgents: if rule.target not in set, return `{ allow: false, reason: "target agent '${rule.target}' not configured" }`
    7. Render payload: `rule.template({ event })` — Handlebars TemplateDelegate called with context `{ event }`
    8. Return frozen `{ allow: true, targetAgent: rule.target, payload: renderedPayload, ruleId: rule.id }`
    9. No match: return frozen `{ allow: false, reason: "no matching rule" }`

    Also export a backward-compatible `evaluatePolicy` function wrapper that delegates to a default PolicyEvaluator. This allows TriggerEngine to continue calling `evaluatePolicy(event, agents)` until Plan 62-02 wires the class instance. Pattern:
    ```typescript
    // Backward-compatible wrapper — Plan 62-02 replaces TriggerEngine's usage
    export function evaluatePolicy(
      event: TriggerEvent,
      configuredAgents: ReadonlySet<string>,
    ): PolicyResult {
      // Pass-through: no rules loaded = deny all except configured agents
      if (!configuredAgents.has(event.targetAgent)) {
        return Object.freeze({ allow: false as const, reason: `target agent '${event.targetAgent}' not configured` });
      }
      return Object.freeze({
        allow: true as const,
        targetAgent: event.targetAgent,
        payload: typeof event.payload === "string" ? event.payload : JSON.stringify(event.payload),
        ruleId: "__default__",
      });
    }
    ```
