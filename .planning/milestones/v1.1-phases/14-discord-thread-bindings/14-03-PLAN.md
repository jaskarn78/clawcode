    **1. Create `src/heartbeat/checks/thread-idle.ts`** following inbox.ts pattern exactly per D-11:

    ```typescript
    const threadIdleCheck: CheckModule = {
      name: "thread-idle",
      async execute(context): Promise<CheckResult> {
        // 1. Get agent config from sessionManager
        // 2. Read thread registry
        // 3. Get bindings for this agent
        // 4. For each binding, check if lastActivity + idleTimeoutMinutes < now
        // 5. If idle, call threadManager.removeThreadSession(threadId) per D-11, D-12
        // 6. Return status with count of cleaned up / remaining
      }
    };
    export default threadIdleCheck;
    ```

    The check needs access to ThreadManager. Since CheckContext only has sessionManager and registry, store the ThreadManager reference on the heartbeat runner or pass it via a closure. The simplest approach: add an optional `threadManager` field to `CheckContext` in `src/heartbeat/types.ts`:
    ```typescript
    export type CheckContext = {
      readonly agentName: string;
      readonly sessionManager: SessionManager;
      readonly registry: Registry;
      readonly config: HeartbeatConfig;
      readonly threadManager?: ThreadManager;  // For thread-idle check
    };
    ```
    Then wire it in the HeartbeatRunner where it builds context (read `src/heartbeat/runner.ts` to find where CheckContext is constructed).

    The idle timeout comes from the agent's config: `agentConfig.threads.idleTimeoutMinutes`. Convert to milliseconds for comparison: `binding.lastActivity + (idleTimeoutMinutes * 60 * 1000) < Date.now()`.

    **2. Wire ThreadManager into daemon.ts:**

    In `startDaemon()`:
    a. After building routingTable (step 5b), create ThreadManager:
       ```typescript
       import { ThreadManager } from "../discord/thread-manager.js";
       import { THREAD_REGISTRY_PATH } from "../discord/thread-types.js";

       const threadManager = new ThreadManager({
         sessionManager: manager,
         routingTable,
         registryPath: THREAD_REGISTRY_PATH,
         log,
       });
       ```

    b. Pass threadManager to the HeartbeatRunner so it can inject into CheckContext. Add `setThreadManager(tm: ThreadManager)` method to HeartbeatRunner (or pass via constructor config).

    c. Add `"threads"` IPC method in `routeMethod`:
       ```typescript
       case "threads": {
         const bindings = await threadManager.getActiveBindings();
         const agentFilter = typeof params.agent === "string" ? params.agent : undefined;
         const filtered = agentFilter
           ? bindings.filter(b => b.agentName === agentFilter)
           : bindings;
         return { bindings: filtered };
       }
       ```

    d. Add threadManager to shutdown sequence -- before `manager.stopAll()`, clean up all thread sessions:
       ```typescript
       const allBindings = await threadManager.getActiveBindings();
       for (const binding of allBindings) {
         try { await threadManager.removeThreadSession(binding.threadId); } catch {}
       }
       ```

    e. Add threadManager to the return object from startDaemon.

    f. Update startDaemon return type to include `threadManager: ThreadManager`.
