  <files>src/skills/linker.ts, src/manager/daemon.ts, src/ipc/protocol.ts</files>

  <action>
    Create `src/skills/linker.ts` (per D-07):
    - Export `linkAgentSkills(agentSkillsDir: string, assignedSkillNames: readonly string[], catalog: SkillsCatalog, log?: Logger): Promise<void>`
    - For each skill name in `assignedSkillNames`, look up the SkillEntry in `catalog`
    - If found, create a symlink at `{agentSkillsDir}/{skillName}` pointing to `entry.path`
    - Use `symlink(entry.path, linkPath)` from `node:fs/promises`
    - If symlink already exists and points to the correct target, skip. If points to wrong target, remove and recreate.
    - If skill name not found in catalog, log a warning and skip (do not error)
    - Use `lstat` + `readlink` to check existing symlink state

    Update `src/manager/daemon.ts` (per D-03, D-09):
    - Import `scanSkillsDirectory` from `../skills/scanner.js`
    - Import `linkAgentSkills` from `../skills/linker.js`
    - After loading config and resolving agents (step 5), resolve `skillsPath` from `config.defaults.skillsPath` (expand `~` to `homedir()`)
    - Call `const skillsCatalog = await scanSkillsDirectory(resolvedSkillsPath, log)`
    - Log catalog size: `log.info({ skills: skillsCatalog.size }, "skills catalog loaded")`
    - For each resolved agent, call `linkAgentSkills(join(agent.workspace, "skills"), agent.skills, skillsCatalog, log)`
    - Pass `skillsCatalog` to `routeMethod` (add parameter)
    - Add return value: include `skillsCatalog` in the returned object from `startDaemon`

    Add IPC `skills` method (per D-09):
    - Add `"skills"` to `IPC_METHODS` array in `src/ipc/protocol.ts`
    - In `routeMethod` in daemon.ts, add `case "skills":` handler:
      - Accept optional `params.agent` (string) to filter by agent
      - Return `{ catalog: Array.from(skillsCatalog.entries()).map(([name, entry]) => ({ ...entry })), assignments: Object.fromEntries(configs.map(c => [c.name, c.skills])) }`
      - If `params.agent` specified, filter assignments to just that agent

    Update `startDaemon` return type to include `skillsCatalog: SkillsCatalog`.
  </action>
