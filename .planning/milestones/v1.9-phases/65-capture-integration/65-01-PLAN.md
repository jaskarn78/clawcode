    Export function `detectInstructionPatterns(content: string): InstructionDetectionResult`:
    - HIGH_RISK_PATTERNS (readonly RegExp array):
      - `/<\s*system\s*>/i` -- system tag injection
      - `/<<\s*SYS\s*>>/i` -- Llama-style system marker
      - `/\[INST\]/i` -- instruction marker
      - `/ignore\s+(all\s+)?previous\s+instructions/i` -- role override
      - `/you\s+are\s+now\s+(a|an)\s+/i` -- role reassignment
      - `/disregard\s+(your|all)\s+(previous\s+)?(training|instructions)/i` -- jailbreak
    - MEDIUM_RISK_PATTERNS:
      - `/repeat\s+your\s+(system\s+)?prompt/i` -- prompt leak
      - `/what\s+are\s+your\s+instructions/i` -- prompt probe
      - `/\[\s*SYSTEM\s*\]/i` -- bracketed system tag
      - `/---\s*\n\s*new\s+conversation\s*\n\s*---/im` -- delimiter abuse (note the 'm' flag for multiline)
    - Iterate both arrays, collect matched pattern .source strings
    - Return Object.freeze({ detected: matched.length > 0, patterns: Object.freeze(matched), riskLevel })
    - riskLevel: "high" if any high pattern matched, else "medium" if any medium matched, else "none"
    - Module must have ZERO imports (pure string matching)

    **1. Create src/discord/capture.ts:**

    ```typescript
    import type { ConversationStore } from "../memory/conversation-store.js";
    import type { Logger } from "pino";
    import { detectInstructionPatterns } from "../security/instruction-detector.js";

    export type CaptureInput = {
      readonly convStore: ConversationStore;
      readonly sessionId: string;
      readonly userContent: string;
      readonly assistantContent: string;
      readonly channelId: string;
      readonly discordUserId: string;
      readonly discordMessageId: string;
      readonly log: Logger;
    };
    ```

    Export function `captureDiscordExchange(input: CaptureInput): void`:
    - Run `detectInstructionPatterns(input.userContent)` BEFORE any storage
    - If detection.detected is true: `log.warn({ risk: detection.riskLevel, patterns: detection.patterns, channel: input.channelId }, "instruction pattern detected in user message")`
    - Build instructionFlags: `detection.detected ? JSON.stringify(detection) : undefined`
    - Call convStore.recordTurn for user turn: { sessionId, role: "user", content: input.userContent, channelId: input.channelId, discordUserId: input.discordUserId, discordMessageId: input.discordMessageId, instructionFlags }
    - Call convStore.recordTurn for assistant turn: { sessionId, role: "assistant", content: input.assistantContent, channelId: input.channelId }
    - Wrap ENTIRE body in try/catch: on error, log.warn({ agent: "capture", error: (err as Error).message }, "conversation capture failed (non-fatal)")
    - NEVER throw -- this function is fire-and-forget
