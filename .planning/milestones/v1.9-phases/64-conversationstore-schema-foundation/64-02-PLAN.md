**Create `src/memory/conversation-store.ts`** following the EpisodeStore pattern but receiving `DatabaseType` directly (not MemoryStore) since ConversationStore does NOT use MemoryStore.insert().

Key implementation details:

1. **Constructor** receives `DatabaseType` (from `store.getDatabase()`). Calls `this.prepareStatements()` to cache all SQL statements.

2. **Prepared statements type:**
```typescript
type ConversationStatements = {
  readonly insertSession: Statement;
  readonly updateSessionEnd: Statement;
  readonly updateSessionCrash: Statement;
  readonly updateSessionSummarized: Statement;
  readonly getSession: Statement;
  readonly listSessions: Statement;
  readonly insertTurn: Statement;
  readonly getTurnsForSession: Statement;
  readonly getTurnsForSessionLimited: Statement;
  readonly getSessionTurnCount: Statement;
  readonly incrementTurnCount: Statement;
  readonly addTokens: Statement;
};
```

3. **Session methods:**
   - `startSession(agentName: string): ConversationSession` -- INSERT with nanoid(), status='active', started_at=now
   - `endSession(sessionId: string): ConversationSession` -- UPDATE status='ended', ended_at=now WHERE status='active'; throw if no rows affected
   - `crashSession(sessionId: string): ConversationSession` -- UPDATE status='crashed', ended_at=now WHERE status='active'; throw if no rows affected
   - `markSummarized(sessionId: string, summaryMemoryId: string): ConversationSession` -- UPDATE status='summarized', summary_memory_id WHERE status IN ('ended', 'crashed'); throw if no rows affected
   - `getSession(sessionId: string): ConversationSession | null` -- SELECT, return null if not found
   - `listRecentSessions(agentName: string, limit: number): readonly ConversationSession[]` -- SELECT ORDER BY started_at DESC LIMIT ?

4. **Turn methods:**
   - `recordTurn(input: RecordTurnInput): ConversationTurn` -- Use a transaction: get current turn_count from session, INSERT turn with turn_index=turn_count, INCREMENT turn_count and total_tokens on session. Uses `nanoid()` for turn ID. Convert boolean isTrustedChannel to 0/1 for SQLite. Default isTrustedChannel to false if not provided.
   - `getTurnsForSession(sessionId: string, limit?: number): readonly ConversationTurn[]` -- SELECT ORDER BY turn_index ASC, optionally with LIMIT

5. **Row conversion helpers:**
   - `rowToSession(row: SessionRow): ConversationSession` -- Object.freeze() with camelCase field mapping
   - `rowToTurn(row: TurnRow): ConversationTurn` -- Object.freeze() with camelCase field mapping. CRITICAL: convert `is_trusted_channel` INTEGER (0/1) to boolean.

6. **State machine enforcement in code:** endSession/crashSession only accept status='active'. markSummarized only accepts status IN ('ended', 'crashed'). Validate by checking rows affected from UPDATE (better-sqlite3 `changes` property). Throw descriptive errors on invalid transitions.

7. **Imports:** `import type { Database as DatabaseType, Statement } from "better-sqlite3"`, `import { nanoid } from "nanoid"`, import types from `./conversation-types.js`.

8. **JSDoc** on class and all public methods per CONVENTIONS.md.
