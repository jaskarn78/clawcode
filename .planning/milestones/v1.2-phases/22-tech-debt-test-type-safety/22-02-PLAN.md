  type: 'result';
  subtype: 'success';

Based on the SDK's `SDKSessionOptions`, include only the fields we use: `model: string`, `cwd?: string`, `systemPrompt?: string`, `permissionMode?: 'default' | 'acceptEdits' | 'bypassPermissions' | 'plan'`.

    2. **SdkSession** -- The session object returned by the SDK. Based on `SDKSession`: `readonly sessionId: string`, `send(message: string): Promise<void>`, `stream(): AsyncGenerator<SdkStreamMessage, void>`, `close(): void`, `on?(event: string, handler: (...args: unknown[]) => void): void` (the `on` is used in wrapSdkSession for error/end events -- add as optional since it may not be on the official type).

    3. **SdkStreamMessage** -- Union type for messages from `session.stream()`. Based on actual usage in wrapSdkSession: `{ type: 'assistant'; content?: string } | { type: 'result'; subtype: string; result?: string; is_error?: boolean; total_cost_usd?: number; usage?: { input_tokens?: number; output_tokens?: number }; num_turns?: number; duration_ms?: number; model?: string }`.

    4. **SdkModule** -- The module shape from `import("@anthropic-ai/claude-agent-sdk")`. An object with: `unstable_v2_createSession(options: SdkSessionOptions): SdkSession`, `unstable_v2_resumeSession(sessionId: string, options: SdkSessionOptions): SdkSession`.

     * SDK version at time of writing: 0.2.97

     */
