Create `src/dashboard/sse.ts` — SseManager class:
- Constructor takes `{ socketPath: string; pollIntervalMs: number; log: Logger }`
- `addClient(res: ServerResponse)`: Adds a response to a Set of active SSE clients. Sets headers `Content-Type: text/event-stream`, `Cache-Control: no-cache`, `Connection: keep-alive`. Writes initial `:keepalive\n\n`. Removes client on close.
- `broadcast(event: string, data: unknown)`: Writes `event: {event}\ndata: {JSON.stringify(data)}\n\n` to all clients.
- `start()`: Starts a setInterval that calls `pollAndBroadcast()` every `pollIntervalMs`.
- `stop()`: Clears interval.
- `pollAndBroadcast()`: Calls `sendIpcRequest(socketPath, "status", {})` and `sendIpcRequest(socketPath, "context-zone-status", {})` in parallel. Merges registry entries with zone data into `AgentStatusData[]`. Broadcasts as `event: agent-status`. On IPC error, broadcasts `event: error` with `{ message: "Daemon not reachable" }`. Catches errors so the interval continues.

Create `src/dashboard/server.ts`:
- `startDashboardServer(config: DashboardServerConfig)` function
- Uses Node.js built-in `http.createServer` (NOT Express — keep dependency-free) per deep_work_rules
- Routes:
  - `GET /` and `GET /index.html` — serve `src/dashboard/static/index.html` (use `import.meta.dirname` to resolve path)
  - `GET /styles.css` — serve CSS with `text/css`
  - `GET /app.js` — serve JS with `application/javascript`
  - `GET /api/events` — SSE endpoint, call `sseManager.addClient(res)`
  - `POST /api/agents/:name/start` — call `sendIpcRequest(socketPath, "start", { name })`, return JSON `{ ok: true }`
  - `POST /api/agents/:name/stop` — same for stop
  - `POST /api/agents/:name/restart` — same for restart
  - `GET /api/status` — one-shot JSON endpoint returning current agent status (for initial page load before SSE connects)
  - All other routes: 404
- Parse URL pathname manually (no framework). For POST routes, extract agent name from URL segments.
- Creates SseManager, starts it, returns `{ server, sseManager, close: () => Promise<void> }`
- `close()` stops SSE manager and closes HTTP server
