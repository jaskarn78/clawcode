  <files>src/memory/episode-archival.ts, src/memory/__tests__/episode-archival.test.ts</files>

  <behavior>
    - archiveOldEpisodes finds episodes older than archivalAgeDays
    - Each archived episode is moved to cold tier via store.updateTier(id, "cold")
    - Archived episode embeddings are deleted from vec_memories (removes from search)
    - Returns EpisodeArchivalResult with count of archived episodes and any errors
    - If no episodes qualify, returns { archived: 0, errors: [] }
    - Errors on individual episodes do not stop the pipeline (partial archival)
  </behavior>
  <action>
    1. Create src/memory/episode-archival.ts:
       - Export type EpisodeArchivalResult:
         ```typescript
         export type EpisodeArchivalResult = {
           readonly archived: number;
           readonly errors: readonly string[];
         };
         ```
       - Export async function archiveOldEpisodes(store: MemoryStore, archivalAgeDays: number): Promise<EpisodeArchivalResult>
       - Implementation:
         - Query: `SELECT id, created_at FROM memories WHERE source = 'episode' AND tier != 'cold' AND created_at < ?` with cutoff date = now minus archivalAgeDays
         - For each qualifying episode in a loop:
           - store.updateTier(id, "cold") to move to cold tier
           - Delete from vec_memories: `DELETE FROM vec_memories WHERE memory_id = ?` (removes from vector search)
           - Increment archived count
         - Wrap each iteration in try/catch, push error message on failure, continue
         - Return frozen EpisodeArchivalResult
       - Follow the same error-tolerant pattern as archiveDailyLogs in consolidation.ts
       - Use pino logger for error logging (import from ../shared/logger.js)

    2. Create src/memory/__tests__/episode-archival.test.ts:
       - Use `:memory:` SQLite database
       - Mock embedder returning random 384-dim vectors
       - Test cases:
         - Returns { archived: 0, errors: [] } when no episodes exist
         - Returns { archived: 0, errors: [] } when all episodes are younger than threshold
         - Archives episodes older than archivalAgeDays (set tier to cold, remove from vec_memories)
         - Does not archive episodes that are already cold
         - Archived episodes no longer appear in vec_memories KNN search
         - Partial archival: if one episode fails, others still archive (mock an error scenario)
         - archivalAgeDays=0 archives all episodes
  </action>
