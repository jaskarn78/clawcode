v1 vs v2 Comparison

Typography
v1: Fraunces serif italic — editorial gravitas, "advanced view vibes"
v2: Cabinet Grotesk geometric sans — modern startup, clean, distinctive. No serif.
Primary accent
v1: Hot pink #ff3366 everywhere — nav active, palette selection, hero metrics
v2: Emerald #10b981 primary, hot pink sparingly (streaming-only indicator). Refined blue for secondary actions.
Mode
v1: Advanced only — single dense power-user view
v2: Basic (default) + Advanced toggle. Basic = glanceable list, not display:none of advanced. Structurally different.
Mobile
v1: Desktop 1440px only — sidebar fixed, no touch targets, no bottom nav
v2: Mobile-first (375px default). Bottom tab nav, collapsible sidebar, touch targets ≥44px at all breakpoints.
Background
v1: #0a0a0e + radial gradient mesh + grain overlay — heavy, theatrical
v2: #0e0e12 (lighter) + solid surfaces — clean, breathable. No grain. No radial mesh.
Ramy indicator
v1: Crown badge absolute-positioned top-right — ornamental
v2: Minimal gold star (★) inline with agent name — elevated, not decorative.
New features
v1: F01-F15 (Tier 1 + Tier 2)
v2: F01-F15 retained + F26 config editor + F27 conversations + F28 task board
Density
v1: 12px body text, tight spacing, many fields per tile
v2: 14-15px body, p-6 card padding, 4 fields max in basic mode. Generous line-height.

Design direction

Soft dark (Option A). Background #0e0e12. Emerald as primary accent. Cabinet Grotesk display. Geist body. JetBrains Mono data.
