## Decisions Made

- **fs.readdirSync over glob.** RESEARCH.md and the plan example used `glob`, but `glob` is not in package.json (production deps include yaml/zod/etc but not glob). Wrote a tiny recursive `walkTsFiles(dir)` walker that handles `__tests__/` filtering inline. Zero dep added; identical behavior.
- **JSDoc historical-rename note rephrasing.** First draft of the new JSDoc said "Renamed from sendToAgent in Phase 999.2 D-RNI-01" — but the static-grep test is repo-wide, so leaving the literal `sendToAgent` in a JSDoc would self-fail Test 1. Rephrased to "Renamed from the previous 'send-to-agent' framing" — conveys the same historical context without the literal substring.
- **DEPRECATED_NAME constructed dynamically.** A test that asserts "no test file contains string X" will fail on itself if the test source contains X literally (the test file IS a test file). Constructed `DEPRECATED_NAME = ["send","To","Agent"].join("")` so the test file passes its own assertion. Documented in a code comment at the top of the test file.

## Deviations from Plan

- **Verification:** All 4 static-grep tests pass; the test file imports zero new packages.
