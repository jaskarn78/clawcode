    - assembleContext with all sources populated returns string with sections in order: identity, hot memories, tool definitions, graph context, discord bindings, context summary
    - assembleContext with identity exceeding 1000-token budget (4000+ chars) truncates to ~4000 chars
    - assembleContext with hotMemories exceeding budget truncates at line boundaries (drops whole bullets, not mid-line)
    - assembleContext with empty identity string omits identity section entirely (no empty header)
    - assembleContext with empty graphContext omits graph section
    - assembleContext with all sources empty returns empty string
    - assembleContext with custom budgets uses those instead of defaults
    - Discord bindings and context summary are included as-is regardless of length (pass-through, no truncation)
    - Section headers (## Key Memories, etc.) are NOT counted against the source budget
    - estimateTokens returns Math.ceil(text.length / 4)
    - exceedsCeiling returns true when assembled string exceeds 8000 tokens (32000 chars)
    - DEFAULT_BUDGETS is frozen with identity:1000, hotMemories:3000, toolDefinitions:2000, graphContext:2000
  </behavior>
  <action>
    RED: Write tests in src/manager/__tests__/context-assembler.test.ts covering all behaviors above. Run tests — all must fail.

    GREEN: Implement src/manager/context-assembler.ts:
    - Export type `ContextBudgets` with readonly fields: identity (number), hotMemories (number), toolDefinitions (number), graphContext (number)
    - Export type `ContextSources` with readonly fields: identity (string), hotMemories (string), toolDefinitions (string), graphContext (string), discordBindings (string), contextSummary (string)
    - Export const `DEFAULT_BUDGETS: ContextBudgets` = Object.freeze({ identity: 1000, hotMemories: 3000, toolDefinitions: 2000, graphContext: 2000 })
    - Export function `estimateTokens(text: string): number` — returns Math.ceil(text.length / 4)
    - Export function `exceedsCeiling(assembled: string, ceiling: number = 8000): boolean` — returns estimateTokens(assembled) > ceiling
    - Internal function `truncateToBudget(text: string, tokenBudget: number): string` — for memories (contains newlines starting with "- "), truncate at line boundaries by dropping trailing bullets; for other sources, truncate at maxChars = tokenBudget * 4 with "..." suffix
    - Export function `assembleContext(sources: ContextSources, budgets: ContextBudgets = DEFAULT_BUDGETS): string`
      - Builds sections array
      - identity: if non-empty, push truncateToBudget(sources.identity, budgets.identity) — NO section header (fingerprint has its own formatting)

      - toolDefinitions: if non-empty, push "## Available Tools\n\n" + truncateToBudget(sources.toolDefinitions, budgets.toolDefinitions)
      - graphContext: if non-empty, push "## Related Context\n\n" + truncateToBudget(sources.graphContext, budgets.graphContext)
      - discordBindings: if non-empty, push as-is (pass-through)
      - contextSummary: if non-empty, push as-is (pass-through)
      - Return sections.join("\n\n")
    - All exported objects frozen with Object.freeze()
