Build the pure, daemon-agnostic search core: config schema, provider clients (Brave + Exa), a URL fetcher, a small Readability adapter that reuses Phase 70's module, and pure tool handlers — plus add `web_search` / `web_fetch_url` to the v1.7 idempotent tool-cache whitelist. Output: A working, unit-tested `src/search/` module + `defaults.search` config field + `web_search`/`web_fetch_url` in the intra-turn cache whitelist. Plan 02 consumes these as-is.
