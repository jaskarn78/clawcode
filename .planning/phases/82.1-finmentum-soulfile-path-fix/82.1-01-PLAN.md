<objective>
Close the single cross-phase wiring gap identified in `v2.1-MILESTONE-AUDIT.md`: `config-mapper.ts` emits `soulFile = join(targetMemoryPath, "SOUL.md")` for every agent, but for the 5 finmentum-family agents `targetMemoryPath = <root>/finmentum/memory/<agentId>` while `workspace-copier` writes SOUL.md to `<root>/finmentum/SOUL.md`. The YAML pointer disagrees with the on-disk location, breaking `clawcode migrate openclaw verify` for these 5 agents. Runtime is unaffected (session-config's 3-branch fallback resolves to `<workspace>/SOUL.md`), but `verify` doesn't know about that fallback.

Fully closes **CONF-01**, **WORK-02**, and **MIGR-04**, which are all `partial` in REQUIREMENTS.md due to this single wiring gap.

Purpose: Align the migrated YAML's `soulFile` / `identityFile` pointers with where `workspace-copier` actually writes those files for shared-basePath agents.
