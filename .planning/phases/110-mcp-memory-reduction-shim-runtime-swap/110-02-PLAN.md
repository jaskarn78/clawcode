Wave 1 schema + loader + observability changes. Three coupled changes that ship together because schema widen alone is dead config without loader read, and loader read with no observability classifier creates a blind spot.

1. **Widen schema enum** `defaults.shimRuntime.{search,image,browser}` from `["node"]` to `["node","static","python"]` (matches Stage 0a code-comment intent at src/config/schema.ts:1629-1630).
