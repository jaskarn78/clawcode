### `src/manager/session-adapter.ts` — transformMcpServersForSdk

Before (lines 1317-1326 — unexported, narrow type):
```typescript
function transformMcpServersForSdk(
  mcpServers?: readonly { readonly name: string; readonly command: string; readonly args: readonly string[]; readonly env: Readonly<Record<string, string>> }[],
): Record<string, { command: string; args: string[]; env: Record<string, string> }> | undefined {
  if (!mcpServers || mcpServers.length === 0) {
    return undefined;
  }
  return Object.fromEntries(
    mcpServers.map((s) => [s.name, { command: s.command, args: [...s.args], env: { ...s.env } }]),
  );
}
```

After:
```typescript
export function transformMcpServersForSdk(
  mcpServers?: readonly {
    readonly name: string;
    readonly command: string;
    readonly args: readonly string[];
    readonly env: Readonly<Record<string, string>>;
    readonly alwaysLoad?: boolean;
  }[],
): Record<string, { command: string; args: string[]; env: Record<string, string>; alwaysLoad?: boolean }> | undefined {
  if (!mcpServers || mcpServers.length === 0) {
    return undefined;
  }
  return Object.fromEntries(
    mcpServers.map((s) => [
      s.name,
      {
        command: s.command,
        args: [...s.args],
        env: { ...s.env },
        // Phase 999.54 (D-01a) — spread-conditional preserves byte-stable
        // deep-equality for fleet entries that don't opt in. `=== true` not
        // truthy-check, matching the `s.optional === true` idiom upstream
        // at loader.ts. Plan 04 pins both branches (set + omitted).
        ...(s.alwaysLoad === true ? { alwaysLoad: true as const } : {}),
      },
    ]),
  );
}
```

Three coupled changes: `function` → `export function`; inline param + return types widened; spread-conditional inside the per-entry map.

## Decisions Made

- **`.optional()` not `.default(false)`** on `mcpServerSchema.alwaysLoad` — keeps the field OMITTED from parsed output when yaml doesn't declare it (matches `description.optional()` / `accessPattern.optional()` precedent two lines above). Critical for byte-stable schema-roundtrip across the 15+ agent fleet.
- **Strict `=== true` guard** (not truthy-check) in the spread-conditional — mirrors the upstream `s.optional === true` idiom in `loader.ts`, avoids accidental coercion from `"true"` strings or `1`/`0`.
- **`true as const` literal** in the emitted value — pins the SDK output field type to `true`, lining up with the `alwaysLoad?: boolean` return-type widening without forcing the broader-typed `boolean` into the emitted shape.
- **Single-chokepoint Rule 3 symmetric edit** — both createSession and resumeSession call the same `transformMcpServersForSdk`. Widening the transform once satisfies the disallowedTools-style symmetric edit precedent without per-call-site changes.
