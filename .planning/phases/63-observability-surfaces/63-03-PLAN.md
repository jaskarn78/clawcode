    **Types:**
    ```typescript
    export type ChainNode = {
      readonly type: "trigger" | "turn" | "task";
      readonly agent: string;
      readonly turnId: string | null;
      readonly taskId: string | null;
      readonly status: string;
      readonly startedAt: number;
      readonly endedAt: number | null;
      readonly durationMs: number | null;
      readonly tokenCost: number;
      readonly depth: number;
      readonly children: readonly ChainNode[];
    };

    export type ChainResult = {
      readonly causationId: string;
      readonly totalTokenCost: number;
      readonly root: ChainNode | null;
      readonly nodes: readonly ChainNode[];
    };
    ```

    **Core function: `discoverAgentTracesDbs(agentsBasePath: string): readonly { agent: string; dbPath: string }[]`**
    - Default agentsBasePath: `join(homedir(), ".clawcode", "agents")`
    - List subdirectories of agentsBasePath
    - For each subdir, check if `traces.db` exists
    - Return array of { agent: subdirName, dbPath: full path to traces.db }

    **Core function: `walkCausationChain(opts: { tasksDbPath: string; agentsBasePath: string; causationId: string }): ChainResult`**
    1. Open tasks.db read-only. Query: `SELECT * FROM tasks WHERE causation_id = ? ORDER BY depth ASC, started_at ASC`. Map to task rows.
    2. Discover agent traces.db files via discoverAgentTracesDbs.
    3. For each agent traces.db, open read-only, query: `SELECT id, agent, started_at, ended_at, total_ms, status, turn_origin FROM traces WHERE turn_origin LIKE '%"causationId":"' || ? || '"%' ORDER BY started_at ASC`. Parse turn_origin JSON via TurnOriginSchema.safeParse. Close DB.
    4. Merge task rows and trace rows into ChainNode objects:
       - Task rows become nodes with type="task", taskId=task_id, agent=target_agent, tokenCost=chain_token_cost
       - Trace rows become nodes with type="turn", turnId=id, agent=agent from row
       - If a trace's TurnOrigin.source.kind === "trigger", mark as type="trigger"
    5. Build tree: root node is the one with depth=0 (or the first trigger-type node). Link children via:
       - Tasks: parent_task_id links to parent task's task_id
       - Turns: TurnOrigin.parentTurnId links to parent turn's id
       - A task's delegated turn: the turn with TurnOrigin.source.kind === "task" and source.id === task_id
    6. Compute totalTokenCost as sum of all task nodes' chain_token_cost.
    7. Return ChainResult with root tree and flat nodes array.

    **Important note on tree building:** The tree structure is:
    - Root trigger turn (or root task)
      - Task 1 (delegated by root turn's agent)
        - Turn for Task 1 (target agent's turn processing the task)
          - Task 2 (delegated by target agent)
            - Turn for Task 2
    
    The linking logic: a task's parent is the turn that delegated it (via caller_agent matching the turn's agent). A turn's parent is the task it's processing (via TurnOrigin.source.id === task_id for kind="task").

    **Core function: `formatChainTree(result: ChainResult): string`**
    1. If result.root is null, return "No chain found for causation_id: {id}"
    2. First line: summary — `Chain {causationId} | {nodeCount} nodes | {formatTokenCount(totalTokenCost)} total tokens`
    3. Recursive render function that builds lines with box-drawing characters:
       - Use ANSI colors: GREEN for complete, RED for failed/timed_out/cancelled, YELLOW for running/pending
       - Each node line: `{prefix}{connector} {type}:{id} ({agent}, {status}, {duration}, {tokenCost} tokens)`
       - Connector chars: `├── ` for non-last children, `└── ` for last child
       - Prefix: `│   ` for each ancestor that is NOT the last child, `    ` for ancestors that ARE last child
       - Truncate turnId/taskId to 12 chars

    Example output:
    ```
    Chain abc123def | 5 nodes | 2.4K total tokens
    trigger:Xk9m2pQ3Rz (acquisition, complete, 4.5s, 450 tokens)
    ├── task:generate-r... (acquisition -> research, complete, 3.4s, 1.2K tokens)
    │   └── turn:Ym8n3qR4Sz (research, complete, 2.1s, 800 tokens)
    └── task:format-out... (acquisition -> studio, complete, 1.8s, 600 tokens)
        └── turn:Zn9o4rS5Tz (studio, complete, 1.5s, 500 tokens)
    ```
