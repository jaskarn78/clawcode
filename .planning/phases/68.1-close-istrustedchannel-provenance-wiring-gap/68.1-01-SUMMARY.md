## Decisions Made

- **Constant `true` at the bridge call site, not threaded `allowed` boolean:** `checkChannelAccess` already early-returns on ACL failure at line 447, so the capture code path is unreachable for untrusted messages. Passing a constant is simpler than threading and clearer in intent — the rationale is captured in an inline comment so future readers understand why a constant is correct.
- **Integration test uses real in-memory `MemoryStore(":memory:")`:** `MemoryStore.migrateConversationTables()` runs on construction and creates both `conversation_turns` AND the `conversation_turns_fts` virtual table with AI/AD/AU triggers, so the integration test exercises the real SQLite FTS5 path without any mocking.
- **TDD two-commit split despite small fix:** RED commit isolates "gap is observable" from GREEN "fix applied" — gives the milestone audit a reproducible proof that the bug existed before the fix landed.
