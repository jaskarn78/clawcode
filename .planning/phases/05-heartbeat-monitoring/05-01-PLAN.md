    - HeartbeatRunner constructor takes { sessionManager, registryPath, config: HeartbeatConfig, checksDir, log }
    - initialize() calls discoverChecks, stores checks
    - start() sets up setInterval at config.intervalSeconds * 1000
    - tick() iterates running agents, for each agent iterates checks, skips checks whose per-check interval hasn't elapsed, executes check with timeout via Promise.race (D-07), stores result in latestResults map, logs to NDJSON file

    - getLatestResults() returns Map<agentName, Map<checkName, { result, lastChecked }>>
    - Sequential execution within tick (D-06) -- no Promise.all

    - Log entries written to `{workspace}/memory/heartbeat.log` as NDJSON (D-08)
