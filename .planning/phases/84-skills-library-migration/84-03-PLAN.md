<objective>
Generate `.planning/milestones/v2.2-skills-migration-report.md` — the operator-facing audit artifact that summarizes every `clawcode migrate openclaw skills apply` run. Report covers per-skill verdict, per-agent linker status, idempotency state, and two cross-cutting invariants: source-integrity hash (sha256 of all ledger source_hash values) and source-tree read-only invariant (mtime comparison of `~/.openclaw/skills/` pre/post-run).

Purpose: Closes SKILL-06. Without a report, the operator has no compact audit trail to show "migration worked, nothing got corrupted, these are the outcomes." Mirrors v2.1's `v2.1-migration-report.md` shape (built by src/migration/report-writer.ts) but scoped to skills only. This plan is Wave 2 alongside Plan 02 — report-writer reads the ledger from Plan 01 + verification output from Plan 02 but does not modify them.
Output: Markdown report generated on every apply, deterministic (re-running against unchanged state produces byte-identical output), atomic temp+rename to survive crashes.
</objective>
