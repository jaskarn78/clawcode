  <behavior>
    LruMap:
    - LruMap(3).set("a",1).set("b",2).set("c",3).set("d",4) evicts "a"; lru.has("a") === false
    - LruMap.get("b") promotes "b" to most-recent; next eviction removes "c" not "b"
    - LruMap.clear() empties the map; lru.size === 0

    DedupLayer:
    - Layer 1 (LRU): calling isDuplicate twice with same sourceId+idempotencyKey returns false then true
    - Layer 2 (debounce): calling debounce() with same sourceId within 5s window returns the pending timer; only the latest event fires after the window
    - Layer 3 (SQLite): insertTriggerEvent returns true on first call, false on duplicate (sourceId, idempotencyKey)
    - purgeTriggerEvents(cutoffMs) deletes rows with created_at < cutoffMs
    - stopAllTimers() clears all pending debounce timers
  </behavior>
  <action>
    Create `src/triggers/dedup.ts` with two exported classes:

    **LruMap<K, V>** (~40 LOC):
    - Constructor takes `maxSize: number`.
    - Internal `Map<K, V>` — uses Map iteration order as LRU ordering.
    - `has(key)`: returns boolean.
    - `get(key)`: if found, delete + re-insert to promote to most-recent, return value. Else undefined.
    - `set(key, value)`: if key exists, delete first. If at capacity, evict oldest (first key via `this.map.keys().next().value`). Then `map.set(key, value)`.
    - `get size()`: return `map.size`.
    - `clear()`: `map.clear()`.

    **DedupLayer**:
    - Constructor takes `{ db: DatabaseType, lruSize: number, defaultDebounceMs: number }` where DatabaseType is the better-sqlite3 Database type.
    - On construction, run DDL:
      ```sql
      CREATE TABLE IF NOT EXISTS trigger_events (
        source_id        TEXT NOT NULL,
        idempotency_key  TEXT NOT NULL,
        created_at       INTEGER NOT NULL,
        UNIQUE(source_id, idempotency_key)
      );
      CREATE INDEX IF NOT EXISTS idx_trigger_events_created_at
        ON trigger_events(created_at);
      ```
    - Prepare statements:
      - `insertStmt`: `INSERT OR IGNORE INTO trigger_events (source_id, idempotency_key, created_at) VALUES (?, ?, ?)`
      - `purgeStmt`: `DELETE FROM trigger_events WHERE created_at < ?`
    - `lru`: LruMap<string, number> with key = `${sourceId}:${idempotencyKey}`, value = timestamp.
    - `timers`: Map<string, NodeJS.Timeout> — key = sourceId, value = pending debounce timer.
    - `pendingEvents`: Map<string, TriggerEvent> — key = sourceId, value = latest event in debounce window.

    Methods:
    - `isLruDuplicate(sourceId: string, idempotencyKey: string): boolean` — check LRU, if found return true. If not found, set in LRU with Date.now(), return false.
    - `debounce(event: TriggerEvent, debounceMs?: number): Promise<TriggerEvent | null>` — returns a Promise that resolves to the event after the debounce window, or null if a newer event replaced it. Uses setTimeout with `.unref()` (Pitfall 3). Key = event.sourceId. If timer already exists for sourceId, clear it and replace pendingEvents entry. Set new timer. On timer fire, delete from timers + pendingEvents, resolve with the event.
    - `insertTriggerEvent(sourceId: string, idempotencyKey: string): boolean` — run insertStmt with created_at = Date.now(). Return `changes > 0` (true if inserted, false if UNIQUE hit).
    - `purgeTriggerEvents(cutoffMs: number): number` — run purgeStmt with cutoffMs. Return `changes` (rows deleted).
    - `stopAllTimers(): void` — clear all pending debounce timers, clear the timers Map and pendingEvents Map.
    - `clearLru(): void` — `lru.clear()`.

    Use `import Database from "better-sqlite3"` and `import type { Database as DatabaseType } from "better-sqlite3"` for the db handle.

    IMPORTANT: All debounce timers MUST call `.unref()` so they don't keep the process alive on shutdown (per RESEARCH Pitfall 3).
  </action>
