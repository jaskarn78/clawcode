    **1. Create `src/triggers/source-registry.ts`:**

    ```typescript
    import type { TriggerSource } from "./types.js";

    export class TriggerSourceRegistry {
      private readonly sources = new Map<string, TriggerSource>();

      register(source: TriggerSource): void {
        if (this.sources.has(source.sourceId)) {
          throw new Error(`TriggerSource already registered: ${source.sourceId}`);
        }
        this.sources.set(source.sourceId, source);
      }

      get(sourceId: string): TriggerSource | undefined {
        return this.sources.get(sourceId);
      }

      all(): readonly TriggerSource[] {
        return [...this.sources.values()];
      }

      get size(): number { return this.sources.size; }
    }
    ```

    **2. Create `src/triggers/engine.ts`:**

    Import: TriggerEngineOptions, TriggerEvent, TriggerSource from types.js. DedupLayer from dedup.js. evaluatePolicy from policy-evaluator.js. TriggerSourceRegistry from source-registry.js. makeRootOriginWithCausation from turn-origin.js. nanoid from nanoid.

    Class `TriggerEngine`:
    - Constructor takes `TriggerEngineOptions` + an optional `configuredAgents: ReadonlySet<string>` parameter.
    - Creates `DedupLayer` using `options.taskStore.rawDb` handle, `options.config.dedupLruSize`, `options.config.defaultDebounceMs`.
    - Creates `TriggerSourceRegistry` as internal field.
    - Stores `turnDispatcher`, `taskStore`, `log`, `config`, `configuredAgents` as private readonly fields.

    Public methods:

    `registerSource(source: TriggerSource): void` — delegates to registry.register(source).

    `async ingest(event: TriggerEvent): Promise<void>`:
    1. Layer 1 — LRU check: `if (this.dedup.isLruDuplicate(event.sourceId, event.idempotencyKey))` log debug + return.
    2. Layer 2 — debounce: `const debounced = await this.dedup.debounce(event)`. If null, return (collapsed into pending timer).
    3. Layer 3 — SQLite UNIQUE: `const inserted = this.dedup.insertTriggerEvent(debounced.sourceId, debounced.idempotencyKey)`. If false, log debug + return.
    4. Policy check: `const decision = evaluatePolicy(debounced, this.configuredAgents)`. If `!decision.allow`, log info with reason + return.
    5. Generate causation_id: `const causationId = nanoid()`.
    6. Build origin: `const origin = makeRootOriginWithCausation("trigger", debounced.sourceId, causationId)`.
    7. Format payload: `const payloadStr = typeof debounced.payload === "string" ? debounced.payload : JSON.stringify(debounced.payload)`.
    8. Dispatch: `await this.turnDispatcher.dispatch(origin, debounced.targetAgent, payloadStr)`.
    9. Update watermark: `this.taskStore.upsertTriggerState(debounced.sourceId, String(debounced.timestamp), null)`.
    10. Log info: `{ sourceId, targetAgent, causationId }`.

    `async replayMissed(): Promise<void>`:
    1. `const maxAge = Date.now() - this.config.replayMaxAgeMs`.
    2. For each source in `this.registry.all()`:
       a. If `!source.poll`, skip.
       b. Read watermark: `const state = this.taskStore.getTriggerState(source.sourceId)`.
       c. `const watermark = state?.last_watermark ?? null`.
       d. If watermark exists and `parseInt(watermark, 10) < maxAge`, log warn + continue.
       e. `const missed = await source.poll(watermark)`.
       f. For each event in missed: `await this.ingest(event)`.
    3. Log info with count of sources replayed.

    `startAll(): void` — for each source in registry: `source.start()`.

    `stopAll(): void` — for each source in registry: `source.stop()`. Then `this.dedup.stopAllTimers()`.

    `updateConfiguredAgents(agents: ReadonlySet<string>): void` — update the internal set (for hot-reload).

    `get registry(): TriggerSourceRegistry` — expose for daemon return value.
