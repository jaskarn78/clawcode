Phase 100 GSD-07: classify `settingSources` + `gsd.projectDir` as NON-reloadable (agent-restart-required) in the config differ, with explicit documentation of WHY (these fields affect SDK session-start baseOptions, not per-turn options). This is the deliberate counterpoint to the 11 prior reloadable classifications (Phases 83/86/89/90/94/95/96): Phase 100 fields require an agent restart, mirroring the v2.5 `memoryPath` pattern at `src/config/types.ts:138-154`.
