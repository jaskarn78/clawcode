    - parseStatPpid(stat): given a /proc/stat line whose comm field contains spaces ("(claude (test-agent))") OR parens ("(node) S 1 ...") OR newlines, returns the correct ppid using lastIndexOf(')'). Throws on malformed input (no closing paren, non-numeric ppid).

    - buildMcpCommandRegexes(serverConfigs): given a `Record<string, {command, args}>`, returns a RegExp that matches if a cmdline string contains either `command` itself OR `command + ' ' + args.join(' ')` substring; escapes regex metacharacters; rejects empty config (throws — boundary validation per coding-style).
    - matchesAnyMcpCommand(cmdline, patterns): tests cmdline string against the union regex.
    - Age helper: `procAgeSeconds({startTimeJiffies, bootTimeUnix, clockTicksPerSec})` — pure math; no syscalls; used by reaper to enforce 5s minimum age (Pitfall 3).
