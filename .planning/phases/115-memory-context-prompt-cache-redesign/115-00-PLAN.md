    For each scenario, time the corresponding trace span name from `src/performance/trace-store.ts`:
    - `cold-start` → `first_token`
    - `discord-ack` → `first_token`
    - `tool-heavy` → `end_to_end` AND `tool_call`
    - `memory-recall` → `end_to_end`
    - `extended-thinking` → `first_visible_token` AND `end_to_end`
