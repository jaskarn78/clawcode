# Plan 117-11: `/verbose` operator Discord toggle

Land the operator-facing `/clawcode-verbose on|off|status` slash command and the SQLite-backed per-channel level state that drives the visibility branching seeded in Plan 117-09. When the channel is `verbose` AND the advisor result is `advisor_result`, the Discord bridge replaces the plain footer with a fenced advice block (≤500 chars).
