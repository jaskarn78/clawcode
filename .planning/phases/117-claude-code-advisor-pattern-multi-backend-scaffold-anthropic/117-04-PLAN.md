# Plan 117-04: `AnthropicSdkAdvisor` + SDK Options wiring + budget observer

## Goal

Wire the SDK's `advisorModel` option into ClawCode's session Options builder; instrument the session-adapter stream loop to detect advisor `server_tool_use` and `advisor_tool_result` blocks and emit `advisor:invoked` / `advisor:resulted` events; install a ground-truth iteration counter at the terminal `result` event; expose `advisorEvents` on `SessionManager` so the Discord bridge (Plan 117-09) can consume.

This plan is the central technical move of Phase 117 — every visible behavior in 117-08/09/11 depends on the events landing correctly.
