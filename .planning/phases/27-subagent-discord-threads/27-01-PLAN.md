    Create src/discord/subagent-thread-types.ts with:
    - SubagentThreadConfig type: { readonly parentAgentName: string; readonly threadName: string; readonly systemPrompt?: string; readonly model?: "sonnet" | "opus" | "haiku" }
    - SubagentSpawnResult type: { readonly threadId: string; readonly sessionName: string; readonly parentAgent: string; readonly channelId: string }

    Create src/discord/subagent-thread-spawner.ts with SubagentThreadSpawner class:
    - Constructor takes: { sessionManager: SessionManager; registryPath: string; discordClient: Client; log?: Logger }
    - spawnInThread(config: SubagentThreadConfig): Promise<SubagentSpawnResult>
      1. Get parent config via sessionManager.getAgentConfig(config.parentAgentName) -- throw if not found
      2. Get parent's first bound channel (parentConfig.channels[0]) -- throw if no channels
      3. Check maxThreadSessions limit from parentConfig.threads (use DEFAULT_THREAD_CONFIG fallback)
      4. Fetch the Discord channel via discordClient.channels.fetch(channelId)
      5. Create a thread via channel.threads.create({ name: config.threadName, autoArchiveDuration: 1440 })
      6. Build session name: "{parentAgent}-sub-{nanoid(6)}"
      7. Build subagent config inheriting from parent:
         - name: sessionName
         - model: config.model ?? parentConfig.subagentModel ?? parentConfig.model
         - channels: [] (subagent does not directly bind to channels -- routing handled by thread manager)
         - soul: (config.systemPrompt ?? parentConfig.soul ?? "") + thread context block
         - schedules: [], slashCommands: [] (subagents don't inherit these)
      8. Start agent session via sessionManager.startAgent(sessionName, subagentConfig)
      9. Persist thread binding with addBinding (reuse existing ThreadBinding type -- agentName is the PARENT, sessionName is the subagent)
      10. Return SubagentSpawnResult

    - cleanupSubagentThread(threadId: string): Promise<void>
      1. Read registry, find binding by threadId
      2. If no binding, return (no-op)
      3. Stop subagent session via sessionManager.stopAgent(binding.sessionName)
      4. Remove binding from registry
      5. Log cleanup -- do NOT delete the Discord thread (per SATH-04: thread remains for history)

    - getSubagentBindings(): Promise<readonly ThreadBinding[]>
      Returns all bindings from the registry (for daemon/IPC visibility)
