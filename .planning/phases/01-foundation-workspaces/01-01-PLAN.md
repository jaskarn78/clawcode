  <action>
    **1. Create src/config/loader.ts:**

    - loadConfig(configPath: string): Promise&lt;Config&gt;
      - Read file at configPath using fs.readFile (throw ConfigFileNotFoundError if not found)
      - Parse YAML using yaml.parse()
      - Validate with configSchema.safeParse()
      - If validation fails, throw ConfigValidationError with formatted issues including agent name context (per Pitfall 4)
      - Return validated Config object (immutable -- never mutate the parsed result)

    - resolveAgentConfig(agent: AgentConfig, defaults: DefaultsConfig): ResolvedAgentConfig
      - Return NEW object (immutable pattern per coding-style.md):
        - name: agent.name
        - workspace: agent.workspace ?? join(expandHome(defaults.basePath), agent.name)
        - channels: agent.channels
        - model: agent.model ?? defaults.model
        - skills: agent.skills.length > 0 ? agent.skills : defaults.skills
        - soul: agent.soul
        - identity: agent.identity
      - Never mutate the input agent or defaults objects

    - resolveContent(value: string): Promise&lt;string&gt;
      - If value contains a newline character, return value as-is (it's inline content)
      - If value matches /^[.~\/]/ AND file exists at expanded path, return file contents
      - Otherwise return value as-is (treat as inline content)

    - resolveAllAgents(config: Config): ResolvedAgentConfig[]
      - Map each agent through resolveAgentConfig with config.defaults
