    - markBootstrapComplete writes the .bootstrap-complete flag file with timestamp

  <action>
    Create src/bootstrap/prompt-builder.ts with:
    - `buildBootstrapPrompt(config: BootstrapConfig): string`
    - Returns a system prompt that:
      1. Explains this is a first-run bootstrap session
      2. Tells the agent its name and bound channels
      3. Instructs the agent: "You are setting up your identity. When you receive your first message, respond by creating your SOUL.md and IDENTITY.md files using the Write tool in your workspace."
      4. Provides a structured template the agent should follow:
         - SOUL.md: Core principles, communication style, areas of expertise, boundaries, personality traits
         - IDENTITY.md: Name, role, display persona, avatar description, vibe
      5. Instructs: "After writing both files, confirm the bootstrap is complete by writing a .bootstrap-complete file."
      6. Notes: "Make the personality distinctive and memorable. Avoid generic AI assistant tropes."
    - The prompt should be detailed enough that the agent can self-bootstrap when it receives any first message (even just "hello")

    - `markBootstrapComplete(workspace: string): Promise<void>`
