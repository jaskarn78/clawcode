## Tasks

### T01 — Bring up test-agent and run advisor trigger

```bash
ssh clawdy 'sudo /opt/clawcode/dist/cli/index.js start test-agent'
```

Wait for ready (check `journalctl -u clawcode -f` for the agent's bootstrap completion).

In Discord channel `1491623782807244880`, send the canonical trigger question:

> *Design a worker pool in Go with graceful shutdown — start by consulting the advisor on the right pattern before writing any code.*

Watch the production log:

```bash
ssh clawdy 'journalctl -u clawcode -f --no-pager 2>&1 | grep -iE "advisor"'
```

**Expected log lines** (from 117.1-01 telemetry):

```
msg="advisor invoked (native server tool fired)" agent=test-agent channel=1491623782807244880 userMessageId=<id>
msg="advisor resulted" agent=test-agent channel=1491623782807244880 variant=advisor_result
```

**Expected Discord behavior:**
- 💭 reaction lands on the triggering user message.
- Agent's response carries `*— consulted advisor (Opus) before responding*` footer.
- NO new thread spawned.

If logs are empty: the native path is NOT firing in production. Diagnose by checking `advisorModel` injection in session-config (verify `dumpBaseOptionsOnSpawn` debug flag on test-agent).

If logs fire but Discord shows no reaction/footer: the bridge subscriber isn't running. Diagnose at the SessionManager instance reference.

**Verify:** Both log lines present AND Discord shows reaction + footer.

### T02 — Exercise `/clawcode-verbose` toggle

In channel `1491623782807244880`:

```
/clawcode-verbose status
```

Expected: ephemeral reply showing `Current level: normal; last changed: <timestamp>`.

```
/clawcode-verbose on
```

Expected: ephemeral reply `Verbose mode ON for this channel — advisor consultations will show advice text inline.`

Re-send the trigger question (or a different substantive request). Expected: response includes the fenced advice block:

```
> 💭 advisor consulted:
> <truncated advice ≤500 chars>
```

```
/clawcode-verbose off
```

Expected: ephemeral reply `Verbose mode OFF — advisor consultations will show 💭 reaction + footer only.`

Re-trigger advisor. Expected: footer-only restored, NO inline advice block.

**Verify:** All four slash interactions land + behavior changes correctly.
