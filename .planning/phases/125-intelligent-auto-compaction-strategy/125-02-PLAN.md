# Phase 125 Plan 02 — Single Extractor Seam + Tier 1 Verbatim + Tier 4 Drop

## Goal

Establish the D-01 single-extractor seam (`src/manager/compact-extractors/`) and collapse both daemon dispatch sites (`daemon.ts:3333` auto-trigger + `daemon.ts:10440` manual IPC) to import from it.
