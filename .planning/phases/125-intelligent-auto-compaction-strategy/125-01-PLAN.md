# Phase 125 Plan 01 — Active-State Header

## Goal

Build the sticky 50-line "ACTIVE STATE" header per BACKLOG-SOURCE §"What gets included in the rolling summary". Persist to `~/.clawcode/agents/<agent>/state/active-state.yaml` AND inject at the top of every heartbeat probe so the agent reads "where are we" cheaply each tick. No compaction changes yet — this is the operator-pain 80% win that ships standalone.
