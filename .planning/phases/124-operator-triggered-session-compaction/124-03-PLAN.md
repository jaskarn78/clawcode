<objective>
Add the Discord admin slash command `/clawcode-compact <agent>` so operators can trigger the Plan 124-01 compaction primitive from a Discord channel without dropping to the CLI. Admin-only; ephemeral responses; reuses Plan 01's `handleCompactSession` via the existing `compact-session` daemon IPC method. Closes SC-2.

Purpose: BACKLOG-SOURCE scenario — Jas in #fin-acquisition watching latency climb wants to compact without opening a terminal.

Output: New inline short-circuit handler + DEFAULT_SLASH_COMMANDS entry + integration test.

**Naming deviation from operator brief:** operator dispatch literally said `/compact <agent>`. The bare name `compact` already exists as a native-CC prompt-channel command (`src/manager/native-cc-commands.ts:68 ALLOWED_NATIVES`) — registering a second `/compact` would be silently displaced by `mergeAndDedupe` at register time (native wins, `slash-commands.ts:1050`). We therefore use the project-canonical `clawcode-` prefix (matches `/clawcode-verbose`, `/clawcode-dream`, `/clawcode-probe-fs`, `/clawcode-fleet`, `/clawcode-usage`). Surfaced as Rule 4 architectural decision in SUMMARY.
</objective>

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-124-08 | Elevation of Privilege | Non-admin invokes `/clawcode-compact` | mitigate | Admin gate via `isAdminClawdyInteraction` BEFORE deferReply (T-02). Reject ephemerally with zero IPC, zero LLM cost. |
| T-124-09 | Information Disclosure | Compaction result leaks into channel | mitigate | Every reply uses `MessageFlags.Ephemeral` — only the invoker sees the result. |
| T-124-10 | Tampering | `agent` argument injection | mitigate | `agent` is forwarded as-is to `handleCompactSession`, which rejects unknown agents via the `getSessionHandle` lookup (AGENT_NOT_RUNNING). No shell exec involved. |
| T-124-11 | Denial of Service | Admin spams `/clawcode-compact` repeatedly | accept | Single-admin host; Phase 105 dispatch queue dedupes in-flight compactions at the daemon edge. |
