# Plan 116-04: Tier 2 deep-dive

## Goal

Wire the Tier 2 inspector surfaces accessed via the agent tile drawer (F11). F11 is the entry point; F12-F15 are content within the drawer's three panels.

## Features

- **F11** Per-agent detail drawer — shadcn `<Sheet>` opening at 60% right-side on desktop, 90% on tablet, full-screen on mobile. Three columns: left = config snapshot, center = Discord transcript (50 turns expandable), right = metrics + memory
- **F12** Per-turn trace waterfall — click any turn in F11 center → custom SVG waterfall of `trace_spans` (context_assemble, first_token, first_visible_token, tool_call.*, typing_indicator, end_to_end); color-coded per SLO; hover shows raw ms + percentile rank
- **F13** Cross-agent IPC inbox viewer — per-agent: pending messages with sender/timestamp/delivery status; fleet-wide 24h message log; heartbeat status per inbox. "ClawCode-unique"
- **F14** Memory subsystem panel — per-agent: tier counts (hot/warm/cold), tier-1 file previews (READ-ONLY in v1), vec_memories vs vec_memories_v2 migration delta, last consolidation timestamps, dream-pass status
- **F15** Dream-pass queue visibility — pending queue depth, next scheduled fire, last 7 dream events (newWikilinks count, promotionCandidates, suggestedConsolidations, themedReflection), D-10 hybrid policy veto window state + veto button with rationale field
