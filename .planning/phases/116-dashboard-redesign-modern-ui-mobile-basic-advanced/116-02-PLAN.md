# Plan 116-02: Tier 1 interactivity

## Goal

Add the four interactive Tier 1 features. F06 is the highest-impact UX feature (Cmd+K palette). F07 is the visual surface for the Finding B fix from 116-00 T01. F09 and F10 add operator-action affordances (pause/resume/rollback for migrations, reconnect for MCP servers).

## Features

- **F06** Cmd+K command palette (shadcn `<Command>` dialog): agent jump, theme toggle, search transcript (placeholder, real search in F27), search memory, recent SLO breaches, recent tool errors
- **F07** Tool latency split panel — Recharts two-bar chart per tool: `tool_execution_ms` (filled emerald bar) vs `tool_roundtrip_ms` (outlined). Visualizes the prompt-bloat tax gap. **Depends on 116-00 T01.** Fallback: if Finding B unfixed, render against `trace_spans` data via existing `/api/agents/:name/tools` endpoint
- **F09** Embedding migration tracker — per-agent phase pill (idle / dual-write / re-embed N% / cutover / v1-dropped); pause/resume/rollback buttons (operator-confirm modal); aggregate fleet bar chart
- **F10** MCP server health panel — per-agent list of MCP servers with status badge (ready/degraded/offline), tool count, last ping; reconnect button (operator-confirm modal)
