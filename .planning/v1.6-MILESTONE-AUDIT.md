## Critical Gaps (Blockers)

### RAG-PDF — PDF ingestion broken

**File:** `src/documents/chunker.ts:80`

```ts
const pdfParse = (await import("pdf-parse")).default;
```

`pdf-parse` v2.4.5 exports `PDFParse` as a named export (class-based API), not a default function. `.default` is undefined.

**Fix:**
```ts
const { PDFParse } = await import("pdf-parse");
const instance = new PDFParse();
const data = await instance.parse(buffer);
```

### A2A-06 — send-to-agent fallback crash

**File:** `src/manager/daemon.ts:869-874`

```ts
} catch (err) {
  const errMsg = err instanceof Error ? err.message : String(err);
  log.warn({ from, to, error: errMsg }, "webhook delivery failed, inbox fallback used");
}
```

`log` is defined inside `createDaemon()` but `routeMethod` has no access to it. When webhook delivery fails, this catch throws `ReferenceError: log is not defined`, breaking the A2A-06 delivery confirmation contract. Inbox is still written but caller gets error instead of `{delivered: false}`.

**Fix:** Pass `log` to `routeMethod` as a parameter, or use `console.warn` as minimal patch.
